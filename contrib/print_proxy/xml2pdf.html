XML2PDF¶

Configuration¶

To properly configure the XML2PDF print service of GISDATEN AG, you need specific configuration in the section print of your yaml file. Please see the standard configuration file as an example, or use the create_standard_yaml script to regenerate your configuration file with the desired options.
